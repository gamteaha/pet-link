# Pet-Link: 데스크톱 캐릭터(컴패니언) 배포 플랫폼 마스터 태스크 목록

이 파일은 여러 플랫폼에 걸친 Dedenne 및 확장 캐릭터 프로젝트의 진행 상황을 추적합니다.

## 🐾 데스크톱 캐릭터 (`/dedenne`)
**목표:** 창과 상호작용하는 다양한 형태(동물, 사람 등)의 데스크톱 컴패니언.
- [x] 기본 창 설정 (PyQt6)
- [x] 투명 및 테두리 없는 창
- [x] 기본 물리 (중력, 창문 위에 앉기)
- [x] 애니메이션 시스템
- [ ] 더 복잡한 상호작용 추가
- [ ] 시스템 트레이 연동
- [ ] 설정 메뉴
- [ ] **[신규]** 인간형 및 다양한 관절을 가진 캐릭터를 위한 범용 애니메이션 구조 설계

## 🌐 웹사이트 (`/dedenne-web`)
**목표:** 캐릭터 다운로드 또는 대화형 기능을 위한 웹 상의 존재감.
